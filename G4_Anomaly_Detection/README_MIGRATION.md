# README - Migration vers G4_Anomaly_Detection

## Pourquoi cette migration ?

Le projet **G4_Anomaly_Detection** remplace **G4_anomaly_engine** avec les améliorations suivantes :

### Nouvelles fonctionnalités
- Détection d'anomalies améliorée
- Système de scoring avancé
- Calcul ROI optimisé
- Meilleure gestion de la base de données

### Architecture modernisée
- Code mieux structuré et modulaire
- Tests unitaires inclus
- Documentation technique complète
- Notebooks d'analyse interactifs

## Fichiers de migration disponibles

- migrate.py : Script de migration automatique
- MIGRATION_GUIDE.md : Guide détaillé de migration
- QUICKSTART.md : Démarrage rapide
